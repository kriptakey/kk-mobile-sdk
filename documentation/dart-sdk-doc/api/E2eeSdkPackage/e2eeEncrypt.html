Encrypt data from mobile side to KMS server

Use this method to encrypt data by using RSA mechanism and send the encrypted data to KMS server. The data is encrypted by using public key sent by the application server during preAuthentication.

Passing object parameter RequestE2eeEncrypt and return ResponseE2eeEncrypt
